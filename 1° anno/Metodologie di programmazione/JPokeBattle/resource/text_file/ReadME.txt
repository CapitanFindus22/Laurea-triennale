To add elements to the file manually use these formats

Pokemon:

	name:Pokemon_name
	Type
	Second Type
	Base Move
	Base Move 2
	Base Hp
	Base Att
	Base Dif
	Base Special Att
	Base Special Dif
	Base Velocity
	Base Xp
	
	Example:
	
	name:Bulbasaur
	Erba
	Veleno
	Ruggito
	Azione
	45
	49
	49
	65
	65
	45
	64

Moves:

	Name
	Type
	Power
	Accuracy
	PP
	Type of move
	
	Example:
	
	Ruggito
	Normale
	
	100
	40
	EnStats
	Att
	1

Level up of pokemon:

	Pokemon_name,level needed move_name, ...
	
	Example:
	Charmander, 9 Braciere, 15 Fulmisguardo, 30 Lacerazione, 38 Lanciafiamme
	
	
Record:

	name rounds_won Pokemon_used

	Example:

	Pippo 15 Squirtle
